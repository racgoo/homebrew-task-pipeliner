# homebrew-task-pipeliner

Homebrew tap for [task-pipeliner](https://github.com/racgoo/task-pipeliner) - A powerful, condition-based task pipeline runner with beautiful CLI output.

## Installation

# Add the tap
brew tap racgoo/task-pipeliner

# Install task-pipeliner
brew install task-pipeliner## Updating
ash
# Update Homebrew's package registry first
brew update

# Then upgrade task-pipeliner
brew upgrade task-pipeliner## Original Repository
